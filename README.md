# project_dio
Foram criadas várias planilhas interativas. O objetivo do documento é servir como apoio para o preenchimento da declaração de imposto de renda. Ele permite preencher as informações e em algumas espaços permite também escolher entre determinadas opções.
Temos 3 abas. 
1. Títular: O usuário preenche as informações pessoais, como nome, CPF, dependente, celular, endereço se é residente no exterior.
2. Informes: O usuário deve preencher as informações relacionadas aos seus bancos e o valor total que tem em cada um deles. No parte superior mostra-se o total que o usuário possui.
3. Notas: O usuário deve preencher os valores de receita de entrada mês a mês
Quando o usuário preencher o documento pode salvar e enviar.
O documento possui botões para navegar entre as diferentes abas
